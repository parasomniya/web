Контроллеры (REST-эндпоинты) для:

- приёма телеметрии (`/api/telemetry/host`, `/api/telemetry/rtk`);
- работы с замесами (`/api/batches...`);
- нарушениями (`/api/violations...`);
- рационами (`/api/rations...`);
- группами (`/api/groups...`);
- зонами компонентов (`/api/storage-zones...`);
- отчётами (`/api/reports/daily...`);
- настройками и уведомлениями.
